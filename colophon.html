Typography

Headings are in Manofa, body copy in Aesthet Nova. Both designed by Mariya Pigoulevskaya for The Northern Block. Fonts are piped in through Adobe Fonts.

Tech & deployment

Built with HTML5 and CSS3 by yours truly, merged to a private GitHub repo, and deployed automatically on Netlify. Static sites make the rockin' 🌎 go 'round.

The wiggle is 90% written by Geoff Graham. Some attributes were tweaked, because I had to.
