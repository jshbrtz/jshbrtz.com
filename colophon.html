Typography

Headings are Archivo Black, by Omnibus-Type. Everything else is Open Sans, by Steve Matteson. Both are piped in from Google Fonts.

The wiggle is 90% written by Geoff Graham. Some attributes were tweaked, because I had to.

Geoff, we may never meet, but that wiggle effect felt like it was written specifically for me.

Tech & deployment

Built with HTML5 and CSS3 by yours truly, merged to a private GitHub repo, and deployed automatically on Netlify.

Static sites make the rockin' 🌎 go 'round.
